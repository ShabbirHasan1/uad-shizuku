UAD-SHIZUKU is application for remove bloat app selectively, get sha256sum of installed foss android apps and install foss apps selectively.
* deblot application with bloat app [list](https://raw.githubusercontent.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation/refs/heads/main/resources/assets/uad_lists.json) selectively
* get sha256sum of app list and scan with virustotal api.
* get android foss offa app [list](https://raw.githubusercontent.com/offa/android-foss/refs/heads/master/README.md) and install through obtanium
* get android foss fmhy app [list](https://raw.githubusercontent.com/fmhy/edit/6eba8bacb09d3baca5703ac4ce5725341a14bd15/docs/mobile.md) and install through obtanium

It supports android mode(apk) with adb wireless shizuku and linux/macos/windows mode(binary) with adb platform-tools.
