# Introduction

UAD-Shizuku is an application for selectively removing bloat apps from Android devices, scanning installed FOSS apps for security, and installing FOSS apps from curated lists.

## Key Features

### Selective Debloating

Remove pre-installed bloatware using the comprehensive [Universal Android Debloater](https://github.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation) app list. Choose exactly which apps to disable or uninstall.

### Security Scanning

Get SHA256 checksums of installed FOSS Android apps and scan them with the VirusTotal API to verify their integrity.

### FOSS App Installation

Browse and install apps from curated FOSS lists:
- [Android FOSS by offa](https://github.com/offa/android-foss) - A comprehensive list of free and open source Android apps
- [FMHY Mobile Apps](https://fmhy.net/) - Community-curated mobile app recommendations

### Cross-Platform Support

- **Android**: Native APK with ADB wireless via [Shizuku](https://shizuku.rikka.app/)
- **Desktop**: Linux, macOS, and Windows support via ADB platform-tools
