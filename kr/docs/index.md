# 소개

UAD-Shizuku는 안드로이드 기기에서 블로트웨어를 선택적으로 제거하고, 설치된 FOSS 앱의 보안을 검사하며, 선별된 목록에서 FOSS 앱을 설치할 수 있는 애플리케이션입니다.

## 주요 기능

### 선택적 블로트웨어 제거

[Universal Android Debloater](https://github.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation)의 포괄적인 앱 목록을 사용하여 사전 설치된 블로트웨어를 제거합니다. 비활성화하거나 제거할 앱을 정확하게 선택할 수 있습니다.

### 보안 검사

설치된 FOSS 안드로이드 앱의 SHA256 체크섬을 확인하고 VirusTotal API로 검사하여 무결성을 검증합니다.

### FOSS 앱 설치

선별된 FOSS 목록에서 앱을 탐색하고 설치할 수 있습니다:
- [Android FOSS by offa](https://github.com/offa/android-foss) - 무료 오픈소스 안드로이드 앱의 포괄적인 목록
- [FMHY Mobile Apps](https://fmhy.net/) - 커뮤니티가 선별한 모바일 앱 추천

### 크로스 플랫폼 지원

- **안드로이드**: [Shizuku](https://shizuku.rikka.app/)를 통한 ADB 무선 연결 지원 네이티브 APK
- **데스크톱**: ADB platform-tools를 통한 Linux, macOS, Windows 지원
